# Personal-Homepage
A homepage for my own personal use.

To use:

- Download it.
- Get an API key from https://api.nasa.gov/#apply-for-an-api-key.
- Make a file "apikey.js" in the js folder.
- Paste 
```
function getKey() {
	return "INSERT_API_KEY_HERE";
}
```
into apikey.js.

- Save apikey.js.
- Enjoy!
